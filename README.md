# Student Activity Tracker - Level 2

## Overview
This is an **interactive Student Activity Tracker** web application built using **HTML, CSS, and JavaScript**. It allows students to track their daily learning activities, mark them as completed, and view real-time progress.  

Unlike Level-1, this version includes **reusable activity toggles**, animated progress updates, and Local Storage support for persistence.

---

## Features
- Display a list of student activities dynamically.
- Toggle activities **Pending ↔ Completed** using checkboxes.
- Real-time **progress bar** and text updates.
- Completed activities are **strikethrough and color-coded**.
- Mobile-responsive layout.
- **Local Storage** saves activity status even after page refresh.

---

## Technologies Used
- **HTML** – Structure
- **CSS** – Styling & animations
- **JavaScript** – Interactivity, state management, Local Storage

---

## How to Use
1. Clone or download this repository.
2. Open `index.html` in a browser.
3. Navigate between **Home → Activities → Progress**.
4. Check or uncheck activities to update their status.
5. View your progress in real-time.
